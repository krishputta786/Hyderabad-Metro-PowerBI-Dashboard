# 🚇 Hyderabad Metro Performance Dashboard

## 📌 Project Overview
An interactive Power BI dashboard developed to analyze Hyderabad Metro operations, passenger travel patterns, revenue performance, station performance, and customer satisfaction using multiple integrated datasets.

---

## 📂 Dataset Structure

| Category | Key Fields |
|----------|------------|
| Customers | Customer ID, Age, Gender, Occupation |
| Transactions | Fare, Final Fare, Ticket Type |
| Trips | Date, Distance, Duration, Delay, Operating Cost |
| Stations | Station Name, Metro Line, Zone |
| Feedback | Rating, Travel Purpose |

---

## 🗂️ Data Model

**Star Schema**

**Fact Tables**
- Trips
- Transactions

**Dimension Tables**
- Customers
- Stations
- Feedback

## 💡 Key Insights

- Identified revenue trends across different metro lines.
- Analyzed weekday vs weekend passenger travel patterns.
- Compared revenue against operating costs.
- Identified top-performing stations based on revenue.
- Monitored customer satisfaction using average ratings.
